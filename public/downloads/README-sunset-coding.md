# Sunset Coding — UI pack

Futuristic sunset vibe for dashboards and coding UIs. Warm gradients, dark base, subtle noise and grid.

## Contents

- `sunset-coding.css` — Theme CSS variables and body background. Use with `data-theme-preview="sunset-coding"` on your `<html>` (and add `.dark` / `.light` for mode).

## Usage

1. Copy `sunset-coding.css` into your project.
2. Import it in your app (e.g. in your root layout or `_app`).
3. Set `data-theme-preview="sunset-coding"` on `<html>` when you want this theme, and toggle `.dark` / `.light` for mode.
